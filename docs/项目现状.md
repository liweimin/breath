## 已完成功能
- 可配置参数：吸气、暂停1、呼气、暂停2、总时长、音量。
- 预设模式：`4-4-4-4`、`4-7-8-0`、`6-0-6-0` 与“自定义”。
- 练习控制：左侧单一图标按钮（播放/停止切换）。
- 入口体验：页面首屏以功能入口为主，不再展示顶部引导说明卡。
- 布局策略：
  - PC 端：左侧固定练习区（动画+操作），右侧滚动参数区（预设+参数+说明）。
  - 移动端：保持单列流式布局。
- 联系方式：页面底部展示联系邮箱和 GitHub 开源地址。
- SEO 基础能力：
  - 页面已补充 `title`、`description`、`canonical`、`robots`、`Open Graph`、`Twitter Card`
  - 已加入 `JSON-LD` 结构化数据（`WebApplication`）
  - 已提供 `robots.txt` 与 `sitemap.xml`
- SEO 内容能力：
  - 页面新增“预设说明与常见问题”内容区，默认折叠展示
  - 页面加入 `FAQPage` 结构化数据和权威参考链接
  - 内容区加入项目由来故事（Vibe Coding + 冥想场景）
- 动效引导：球体按阶段变化，中心显示秒级倒计时，底部叠加总剩余进度条。
- 声音引导：
  - 吸气：合成提示音
  - 暂停：合成提示音
  - 呼气：采样音 `assets/audio/exhale-light-short.mp3`
- 导出功能：导出完整练习为 `WAV`，文件名格式 `吸-停1-呼-停2-breath.wav`（示例 `4444breath.wav`）；当前导出入口按钮已隐藏，导出逻辑保留。
- 背景氛围：导出与实时播放均包含雨声白噪合成层。
