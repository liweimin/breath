# 环境与命令

## 本地运行
在项目目录启动本地 HTTP 服务（推荐）：

```bash
cd d:\code\vb
python -m http.server 5500 --bind 0.0.0.0
```

访问地址：
- 本机：`http://127.0.0.1:5500`
- 局域网手机：`http://<你的局域网IP>:5500`

说明：
- 直接双击 `index.html`（`file://`）可能触发浏览器音频限制。
- 需要稳定体验与线上一致时，优先用本地 HTTP。

## Git 常用命令（当前项目）
```bash
cd d:\code\vb
git status
git add .
git commit -m "你的提交说明"
git push
```

说明：
- 仓库已配置远程 `origin` 指向 GitHub。
- `main` 已与 `origin/main` 建立跟踪关系，后续直接 `git push` 即可。

## Git 代理（当前机器已用过）
设置代理（当前使用端口 `7897`）：

```bash
git config --global http.proxy http://127.0.0.1:7897
git config --global https.proxy http://127.0.0.1:7897
```

查看代理：

```bash
git config --global --get http.proxy
git config --global --get https.proxy
```

取消代理：

```bash
git config --global --unset http.proxy
git config --global --unset https.proxy
```

## Vercel 部署参数（本项目）
- Framework Preset：`Other`
- Root Directory：`./`
- Build Command：留空
- Output Directory：`.`（或留空使用静态默认）

说明：
- 这是纯静态项目，不需要 `dist` 构建目录。
- 推送到 GitHub `main` 后，Vercel 会自动触发新部署。

## 可选命令：生成内嵌音频数据
```powershell
powershell -ExecutionPolicy Bypass -File scripts\generate-audio-data.ps1
```

说明：
- 该命令会在项目根目录生成 `audio-data.js`（默认不提交，已在 `.gitignore` 中）。
